= 從 Deploy 中刪除ONTAP Select映像
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
當不再需要ONTAP Select映像時，您可以從 Deploy 管理實用程式實例中刪除它們。


CAUTION: 您不應刪除叢集正在使用的任何ONTAP Select映像。

.關於此任務
您可以刪除叢集目前未使用或計劃在未來叢集部署中使用的舊ONTAP Select映像。

.步驟
. 使用管理員 (admin) 帳戶透過 SSHSign inDeploy 實用程式 CLI。
. 顯示由 Deploy 管理的叢集並記錄正在使用的ONTAP映像：
+
[source, cli]
----
cluster show
----
+
請注意每種情況下的版本號碼和虛擬機器管理程式平台。

. 啟動 Bash shell：
+
[source, cli]
----
shell bash
----
. 顯示所有可用的ONTAP Select映像：
+
[source, cli]
----
ls -lh /opt/netapp/images
----
. 可選擇使用虛擬機器管理程式主機刪除ONTAP Select映像。
+
[role="tabbed-block"]
====
.ESXi 範例
--
[source, asciidoc]
----
rm -r /opt/netapp/images/DataONTAPv-9.3RC1-vidconsole-esx.ova
----
--
.KVM 範例
--
[source, asciidoc]
----
rm -r /opt/netapp/images/DataONTAPv-9.3RC1-serialconsole-kvm.raw.tar
----
--
====
